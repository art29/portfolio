Remote Printer

Remote Printer is a Raspberry App made to control 3D Printer with the Google Assistant. This helps for different use cases : Maintenance (Easier with hands free control), Accessibility (People with disabilities can do some actions more easily) and more !

Technologies

Remote Printer uses Python with Flask to control everything with the help of Ngrok and OctoPrint to control the 3D Printer and Dialogflow to control de Voice Assistant side of the app.
